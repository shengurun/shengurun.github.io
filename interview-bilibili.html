Bilibili 面试题

以下是本人参加了 5 轮 B 站流媒体开发的面试题汇总。

项目相关

  1. 请简单介绍一下你上一次做的项目。项目的架构是怎样的？部署又是怎样实现的？

  2. 一个项目的吞吐量、QPS 你通常是怎样测定的？

  3. 通过 top 命令查看到一个 Python 服务占用了大量内存，又通过一些手段获取到了 profile，但发现 profile 查到的内存和 top 命令展现的对应不上。这时候，应该如何排查？

常用概念及协议

  1. 请介绍一下 进程、线程 和 协程。

  2. 你了解哪些 进程间通（IPC） 的方式？

  3. 阐述一下 Linux 中的几种 IO 模型。

  4. IO 模型中，IO 多路复用 的实现方案有哪些？select、poll 和 epoll 的相同点和不同点有哪些？

  5. 简述一下 TCP/IP 协议中的 四次挥手 以及 C 端和 S 端的状态变化。

  6. TCP 的包头 里有哪些内容？

  7. 简述一下 TCP 和 UDP 的区别。

  8. 如何使用 UDP 实现可靠传输？

  9. TCP 为了实现可靠传输的技术有哪些？

虚拟化

  1. Docker 中，容器（container） 和 **镜像（image）**的区别是什么？

  2. Docker 镜像的本质是什么？

  3. 容器是如何实现 资源隔离 的？

NoSQL 相关

  1. Redis 有哪些数据结构？

  2. Redis 的使用场景有哪些？

  3. Redis 有没有事务？

  4. 能说一下 Redis 高可用的方案吗？

  5. MongoDB 的特点有哪些？

  6. MongoDB 没有事务，那怎么做到原子操作？

  7. 如果让你来做，你会如何设计实现 MongoDB 的事务？

  8. 介绍一下 MongoDB 的索引。可以不创建索引吗？

  9. MongoDB 的磁盘占用大小和存储的文档数据大小有怎样的关联关系？

编程语言相关

  1. 了解过 Tornado 的异步原理和底层机制吗？

  2. 简单说下 Golang 和 Python 的不同之处。

  3. 了解过 Golang 的 编译过程 吗？

  4. 了解过 Python 的 解释过程 吗？

  5. Python 的 元类 是什么？

  6. Python 中 单例模式 的实现方案有哪些？

场景设计

  1. 现在有一个需求，想要统计 5 分钟内用户登录次数，预设每秒有 5000 个用户登录（含重复登录的用户）。数据库和系统应该如何设计？

其他

  1. 介绍一下你参与过的开源项目，这段经验对你有何影响？
